# Plotset Cli

## How to use

- Install the package with `npm i -g plotset-cli`

## Login
`plotset-cli login`

## project
`plotset-cli newProject project name` 

create project template folder 

## deploy
- step1 `cd projectName`

- step2 `npm i`

- step3 `npm run build`

- step4 `plotset-cli deploy`
